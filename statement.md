# Welcome!

This Python template lets you get started quickly with a simple one-page playground.

```python runnable
massa = 95;
g = 10; # aceleracao da gravidade
a = 0.5; # aceleracao do elevador

pr = massa*g;
pm = massa*a + massa*g;

percentual = (pm-pr)/pr;

print("A diferenca percentual e igual a", percentual*100);
```
